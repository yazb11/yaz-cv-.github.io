## Local Development
```bash
# Clone the repository
git clone https://github.com/yazb11/yazb11.github.io.git

# Navigate to project directory
cd yazb11.github.io

# Run locally (using Python's HTTP server)
python -m http.server 8080
```

Visit `http://localhost:8080` in your browser.

## Updating Content
To update the website content, edit these files:
- `index.html` - Main content in both languages
- `styles.css` - Visual styling
- `script.js` - Language switching functionality
